Cloud sync

Keeps this list on Supabase (free tier) so phone and desktop share the same tasks. Your sync ID acts like a password — use a long random value and store it somewhere safe.

One-time setup in Supabase

Create a project at supabase.com, open SQL Editor, run:

create table devmike_sync (
  id text primary key,
  payload jsonb not null,
  updated_at timestamptz not null default now()
);
alter table devmike_sync enable row level security;
create policy "devmike_rw" on devmike_sync
  for all using (true) with check (true);

Then copy Project URL and anon public key from Settings → API.
